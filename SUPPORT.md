# Support

Use [GitHub Discussions](https://github.com/AI-Jerry/review-software-security/discussions) for usage questions when enabled, and [GitHub Issues](https://github.com/AI-Jerry/review-software-security/issues) for reproducible non-sensitive defects and feature proposals.

This is a community project with no guaranteed response time or service-level agreement. Do not post secrets, personal data, production evidence, or private code. Report vulnerabilities through the private process in [SECURITY.md](SECURITY.md).
